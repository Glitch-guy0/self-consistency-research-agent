✅ Decision 1 — Configurable Agent Count

Default 3, configurable via agentCount in factory config. No hard cap.

✅ Decision 2 — Confidence Scoring

Validation uses agreement-strength analysis. When answers converge → synthesized output. When they diverge → confidence scores + differing results shown.

✅ Decision 3 — Per-Agent LLM Providers

Each research agent gets its own {baseUrl, apiKey, model}. GPT-4o, Claude, Gemini — genuine independence.

✅ Decision 4 — Two-Phase Factory

registerResearchAgent(providerConfig) builds the roster, spawnAll() dispatches. The factory is a roster manager.

🔶 Decision 5 — Connection Pooling Deferred

Ship without it, measure, decide later. Not a correctness concern.
