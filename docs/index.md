## Project Overview

An experimental TypeScript framework implementing the self-consistency AI methodology. Spawns multiple parallel LLM research agents to independently investigate a query, then validates and synthesises their outputs into a single answer with confidence scoring.

## Quick Reference

- **Tech Stack:** Node.js (ESM), TypeScript 6.0.3, OpenAI SDK ^6.45.0, Zod ^4.4.3, Chalk ^5.6.2
- **Entry Point:** `main.ts`
- **Architecture Pattern:** Hexagonal (Ports & Adapters)
- **Storage:** In-memory KV cache (no database)
- **LLM Provider:** OpenAI Responses API (configurable base URL/model)

## Getting Started

### Prerequisites

- Node.js v18+
- OpenAI-compatible API endpoint with API key

### Setup

```bash
npm install
cp .env.example .env
# Edit .env with your BASE_URL, MODEL, API_KEY
```

### Run Locally

```bash
npm run dev
```

### Run Typecheck

```bash
npm run typecheck
```
